Les commentaires du code

Dans cet exercice, nous apprenons à écrire des commentaires dans notre code,
ce qui permet de se repérer ainsi que de se relire.
Il est donc tout-à-fait normal que rien de spécial n'apparaît dans cette page web.

Pour afficher un commentaire dans un code, il faut poser la balise ouvrante : "<!--" et la balise fermante : "-->"
